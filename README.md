# 📊 Student Report Card App

A React-based web application to manage student records with dynamic UI updates and real-time calculations.

---

## 🚀 Features

- ➕ Add new student with subject-wise marks  
- ✏️ Edit marks directly in the table  
- 🗑️ Delete student  
- 📊 Automatic Total & Average calculation  
- ✅ Pass / ❌ Fail status (based on average ≥ 40)  
- 📈 Summary Dashboard:
  - Total Students  
  - Passed Students  
  - Failed Students  
- ⚡ Instant UI updates using React state  
- 🎨 Clean UI using Tailwind CSS  

---

## 🧠 Logic Used

- `useState` for managing dynamic data  
- Conditional rendering for Edit mode  
- Array methods:
  - `map()` → display students  
  - `filter()` → count pass/fail  
- Derived values:
  - Total = sum of all subjects  
  - Average = Total / 4  

---

## 🛠️ Tech Stack

- React (Vite)
- Tailwind CSS
- JavaScript (ES6)
